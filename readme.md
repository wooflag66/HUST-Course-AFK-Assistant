# HUST-Course-AFK-Assistant



## 功能

- **2 倍速**：自动锁定 2.0x 播放速度，更高倍速会影响后台数据校验。
- **静音播放**：默认开启静音。
- **自动跳转**：视频播放完毕后，自动定位跳转至下一节。



## 运行方法

1. 安装Tampermonkey浏览器扩展

    [篡改猴 - Chrome 应用商店](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo?hl=zh-CN)。

2. 打开开发者模式以允许脚本运行

   [点击这里查看指引](https://www.tampermonkey.net/faq.php#Q209)

3. 点击下面的链接安装本仓库的脚本

   [点击这里安装脚本](https://greasyfork.org/zh-CN/scripts/569410-华科课程平台视频挂机助手)

4. 进入课程视频界面，等待三秒即可进入挂机状态。



- **关于挂机方式**：建议将刷课标签页缩小窗口，显示在屏幕上。保证刷课标签页不被最小化且不被其他窗口覆盖，这样脚本的运行稳定性最高。
- **关于题目**：章节测验任务点不会变绿。请在视频播完后手动完成。
